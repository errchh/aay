# aay – All Around You

Agent skills for retrieving movie information in Hong Kong.

## Skills

- **hk-cinema-now-playing**: Fetches currently screening films in Hong Kong cinemas via TMDB API

## Setup

Add your TMDB API key to `~/.openclaw/.env`:

```
TMDB_API_KEY=your_api_key_here
```

## Usage

Invoke the skill when querying about movies currently playing in Hong Kong.
