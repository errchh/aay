# aay – All Around You

Agent skills for retrieving movie information in Hong Kong.

![Movie List](assets/movie-list.png)

![Movie Details](assets/movie-details.png)

## Skills

- **hk-cinema-now-playing**: Fetches currently screening films in Hong Kong cinemas via TMDB API

## Setup

Add your TMDB API Read Access Token to 

`~/.openclaw/.env` OR 

`.env` in the project directory:

```
TMDB_API_KEY=your_api_key_here
```
