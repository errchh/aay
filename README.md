# aay – All Around You

Agent skills for Hong Kong cinema movie ticketing. 

## Skills

**hk-cinema-now-playing**: Meet your new AI cinema concierge. 🍿 

User Flow:

- Ask what’s on -> 
- Chat film overviews -> 
- Pick what takes your fancy -> 
- Agent finds cinemas & times -> 
- Select your slot -> 
- Straight to booking 

No scrolling, just chatting. 🎟️

## API Keys

Copy `.env.example` to `.env` and add:

- `TMDB_API_KEY` – TMDB Read Access Token
- `RAPIDAPI_KEY` – RapidAPI key (IMDB data)
- `X-Api-Key` – International Showtime API key
